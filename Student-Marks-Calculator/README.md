# Student Marks Calculator

## About the Project

This is a simple Python project that calculates a student's total marks, percentage, grade, and pass/fail status.

The program allows the user to:

* Enter student name
* Enter number of subjects
* Input subject names and marks
* Calculate total and percentage
* Display grade and result

---

## Features

* Dynamic subject input
* Calculates total marks
* Calculates percentage
* Displays pass/fail status
* Assigns grades automatically

---

## Technologies Used

* Python

---

## How to Run the Project

1. Install Python on your computer
2. Save the program as:

```text
student_marks.py
```

3. Open terminal or command prompt
4. Run the file using:

```bash
python student_marks.py
```

## Grading System

| Percentage   | Grade |
| ------------ | ----- |
| 90 and above | O     |
| 80 - 89      | A     |
| 70 - 79      | B     |
| 60 - 69      | C     |
| 50 - 59      | D     |
| Below 50     | F     |
