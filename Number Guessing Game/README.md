# Number Guessing Game

## About the Project

This is a simple Number Guessing Game developed using Python.

The program randomly generates a secret number between 1 and 100, and the user has to guess the correct number.

The game provides hints such as:

* Too high
* Too low

until the correct number is guessed.

---

## Features

* Random number generation
* User input handling
* Hint system for guesses
* Unlimited attempts until correct guess
* Beginner-friendly game logic

---

## Technologies Used

* Python
* `random` module

---

## How to Run the Project

1. Install Python on your computer
2. Save the program as:

```text
number_guessing_game.py
```

3. Open terminal or command prompt
4. Run the file using:

```bash
python number_guessing_game.py
```
