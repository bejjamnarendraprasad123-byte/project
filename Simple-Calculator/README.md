# Simple Calculator Using Python

## About the Project

This is a simple command-line calculator project developed using Python.

The calculator allows users to perform basic arithmetic operations through a menu-driven program.

---

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Continuous calculations using loop
* Exit option
* Handles division by zero
* Validates invalid menu choices

---

## Technologies Used

* Python

---

## How to Run the Project

1. Install Python on your computer
2. Save the program as:

```text
calculator.py
```

3. Open terminal or command prompt
4. Run the file using:

```bash
python calculator.py
```
