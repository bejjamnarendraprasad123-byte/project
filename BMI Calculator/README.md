# BMI Calculator

## About the Project

This is a simple BMI (Body Mass Index) Calculator developed using Python.

The program takes the user's:

* Height in meters
* Weight in kilograms

and calculates the BMI using the standard BMI formula.

It also displays the BMI category based on the calculated result.

---

## Features

* Calculates BMI using user input
* Displays BMI up to 2 decimal places
* Categorizes BMI result
* Simple and beginner-friendly project

---

## BMI Categories

| BMI Range    | Category      |
| ------------ | ------------- |
| Below 18.5   | Underweight   |
| 18.5 - 24.9  | Normal Weight |
| 25 - 29.9    | Overweight    |
| 30 and above | Obese         |
