# Advanced Password Generator

## About the Project

This is an advanced password generator project developed using Python.

The program generates secure random passwords based on user preferences such as:

* Uppercase letters
* Lowercase letters
* Numbers
* Symbols

It also checks password strength and can generate multiple passwords at once.

---

## Features

* Custom password length
* Include/exclude uppercase letters
* Include/exclude lowercase letters
* Include/exclude numbers
* Include/exclude symbols
* Generates multiple passwords
* Password strength checker
* Prevents weak password combinations
* Input validation using exception handling

---

## Technologies Used

* Python
* random module
* string module

## How to Run the Project

1. Install Python on your computer
2. Save the program as:

text
advanced_password_generator.py


3. Open terminal or command prompt
4. Run the file using:

bash
python advanced_password_generator.py
